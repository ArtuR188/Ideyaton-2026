Як користуватися:

1. Підготовка (Tab 1)
Вкажіть позицію та рівень кандидата (наприклад, Middle Python Developer).

Завантажте резюме у форматі PDF.

Натисніть "🔥 Проаналізувати CV". AI згенерує список індивідуальних технічних питань.

2. Проведення (Tab 2)
Задавайте питання кандидату.

Вписуйте відповіді у відповідні поля та натискайте "Оцінити відповідь".

Ви одразу побачите оцінку від 1 до 10 та технічний аналіз помилок або пропущених моментів.

Після питань та віддповідей можете глянути Tab 3 там будуть результати

3. Live Co-pilot (Tab 4) 
Якщо ви хочете отримувати підказки під час живої розмови, натисніть "🔴 Слухати".

AI розпізнає мову та миттєво запропонує наступне влучне питання, базуючись на тому, що щойно сказав кандидат.

4. Результати (Tab 3)
Перегляньте Карту успішності (графік балів).

Натисніть "🤖 Сформувати опис" для отримання повного портрета кандидата (сильні та слабкі сторони).

Отримайте фінальну рекомендацію щодо найму.
